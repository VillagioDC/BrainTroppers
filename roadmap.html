Storm Trooper MVP Roadmap

A 30-day plan to build an AI-powered brainstorming platform

Phase 1: Foundation Setup
Establish core infrastructure
Day Task
✅ Set initial name, logo and check namecheap namesilo availability
✅ Initialize GitHub repo, setup VSCode workspace, configure CI/CD via Netlify
❌ Define MongoDB schema for projects/nodes, validate with mock data
❌ Setup Express.js backend server with basic routing and middleware
❌ Connect MongoDB to backend using Mongoose ODM
❌ Create authentication framework (OAuth prep: Google/FB placeholders)
Phase 2: Core Backend Development
Build APIs for map generation and node manipulation
Day Task
❌ Integrate OpenAI API for LLM-driven map generation
❌ Implement /create-map endpoint for user topics → JSON maps
❌ Develop /update-node endpoint for actions: approve, rewrite, expand
❌ Add bidirectional edge logic for non-hierarchical connections
❌ Test API endpoints with Postman (JSON schema validation)
Phase 3: Frontend UI Development
Create interactive map canvas and user interface
Day Task
❌ Setup React.js project structure with D3.js visualization
❌ Design landing page and map canvas layout
❌ Implement draggable nodes and edges with D3.js physics
❌ Enable node creation/editing (text input, type selection)
❌ Build context menu for node actions (approve, rewrite, expand)
Phase 4: Integration & AI Features
Connect frontend to backend and enhance AI functionality
Day Task
❌ Link frontend to backend APIs for saving/loading maps
❌ Implement LLM-driven node rewriting (prompt + instruction)
❌ Add "Expand Details" feature (LLM generates sub-nodes)
❌ Optimize LLM prompts for consistency (JSON validation)
❌ Add filtering/search for large maps (tags, approval status)
Phase 5: Advanced Features
Add collaboration, export, and productivity tools
Day Task
❌ Build PDF export with jsPDF (basic layout: map preview only)
❌ Enhance PDF export with detailed node metadata per page
❌ Implement real-time collaboration with Socket.IO
❌ Add keyboard shortcuts (Ctrl+Enter, Undo/Redo)
❌ Develop project dashboard for saving/loading multiple maps
Phase 6: Testing & Optimization
Ensure stability, performance, and user readiness
Day Task
❌ Conduct usability testing with 5 users (record feedback)
❌ Fix critical bugs and optimize load times (lazy loading)
❌ Write technical documentation (APIs, schema, deployment guide)
❌ Record demo video and prepare marketing assets
❌ Deploy MVP and launch publicly
